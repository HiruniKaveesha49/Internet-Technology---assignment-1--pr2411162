Welcome to the HTML Fundamentals Reference Website

This website is a reference guide for beginners who are learning Internet Technologies and HTML. It explains the basic concepts of the Internet, how the Internet works, the role of web browsers, the importance of DNS, and commonly used HTML tags.

HTML is the foundation of web development. It is used to structure content on web pages using elements such as headings, paragraphs, lists, links, images, tables, and forms. Understanding HTML is important for building well-structured, accessible, and search-engine-friendly websites.

What is the Internet?

The Internet is a global network of connected computers, servers, mobile devices, and other digital systems. These devices communicate with each other using standard communication rules called protocols.

The Internet allows users to share information, send messages, access websites, stream videos, use online services, and communicate across the world. It connects people, businesses, schools, governments, and organizations through digital communication.

In simple words, the Internet is a worldwide system that allows devices to exchange information.

How the Internet Works

The Internet works using a system of connected devices, servers, and communication protocols.

When a user opens a website, the user’s device acts as the client. The client sends a request to another computer called a server. The server stores website files and sends the requested information back to the client.

This process is called the client-server model.

For example, when someone types a website address into a browser, the browser sends a request to the website’s server. The server responds by sending the web page files. The browser then displays the website to the user.

The Internet uses protocols such as HTTP and HTTPS to transfer web pages. HTTP stands for HyperText Transfer Protocol. HTTPS is the secure version of HTTP and protects data during communication.

Web Browsers

Role of Web Browser

A web browser is software used to access and view websites.

Examples of web browsers include Google Chrome, Mozilla Firefox, Microsoft Edge, Safari, and Opera.

The browser sends requests to web servers, receives website files, and displays them as web pages. It reads HTML documents and shows the content in a user-friendly way.

Without web browsers, users would not be able to easily view and interact with websites.

What Can a browser Display

Text and HTML Content

Browsers read HTML files and display headings, paragraphs, lists, and links.

Images and Media

Browsers can display images, videos, audio, and other media.

Forms and Interaction

Browsers allow users to fill forms, click buttons, and interact with web pages.

Security Features

Browsers support HTTPS, bookmarks, downloads, extensions, and developer tools.
